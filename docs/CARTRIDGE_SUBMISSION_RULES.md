# Cartridge Submission Rules

PixelForge welcomes tiny games, learning cartridges, templates, experiments, and polished 369 PocketGames candidates.

## Every cartridge should include

- a short README,
- a clear title,
- one-sentence pitch,
- run instructions,
- a tiny complete loop,
- notes about any assets used.

## Good cartridge size

Aim for:

```text
1 room + 1 mechanic + 1 feeling + 1 ending
```

## 369 PocketGames candidate checklist

- playable offline,
- mobile-readable text,
- touch-friendly buttons,
- no ads,
- paid once,
- no subscription model,
- first minute feels charming.
