# Make Your First PixelForge Cartridge

Start small:

```text
1 room + 1 mechanic + 1 feeling + 1 ending
```

## Steps

1. Copy `games/_template/` to a new folder.
2. Rename the package and title.
3. Change the starter scene.
4. Add one tiny loop.
5. Add one ending.
6. Run it locally.
7. Share it for feedback.

A finished tiny thing beats an unfinished giant thing.
