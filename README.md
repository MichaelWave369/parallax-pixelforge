# Parallax PixelForge

Open-source retro game creation studio for building tiny cartridge-style games, sample adventures, and 369 PocketGames.

PixelForge Studio is the free/open-source forge.
Sample cartridges teach the format.
Community cartridges grow the arcade.
369 PocketGames is the polished mobile/premium label.

## Current launch target

v5.3 Public GitHub Launch Kit.

## 369 PocketGames promise

Small, complete games with no ads, no subscriptions, no loot boxes, no energy timers, and no predatory in-app purchases. Signature target price: $3.69.
