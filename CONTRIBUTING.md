# Contributing to Parallax PixelForge

Thanks for helping build the tiny-world forge.

PixelForge is for small, heartfelt, local-first retro game worlds that people can actually finish.

## Project lanes

```text
PixelForge Studio      = open-source forge
Sample cartridges      = runnable learning examples
Community cartridges   = user-created tiny worlds
369 PocketGames        = polished mobile/premium label
```

## Good first contributions

- Improve beginner docs.
- Build a tiny cartridge from `games/_template/`.
- Add validation checks.
- Improve mobile readability.
- Write tests for cartridge manifests.
- Open an issue with a clear, kind bug report.

## Cartridge rule

Start small:

```text
1 room + 1 mechanic + 1 feeling + 1 ending
```

## Rights rule

Only contribute material you created, own, or have permission to use. Do not add ripped sprites, copyrighted music, franchise characters, unclear web images, or private material.

## Pull requests

Before opening a PR, run:

```bash
npm run github:preflight
```

For cartridge submissions, include:

- a README,
- rights notes,
- how to run it,
- screenshots if available,
- a clear status: learning example, playable community cartridge, or 369 PocketGames candidate.
